# DOM and Event Assignment
## Overview
This project is a simple DOM manipulation assignment built using HTML, CSS, and JavaScript.
Users can:
- Change box colors by clicking them
- Enter their name and display greeting
- Reset all changes using reset button

# Features
- DOM Manipulation
- Event Listeners
- Dynamic Greeting
- Background Color Change
- Reset Functionality

# Technologies Used
- HTML
- CSS
- JavaScript

# Functionality
## Box Click
Clicking a box changes its background color based on its text.
Example:
- Red → red background
- Blue → blue background

## Greeting
User enters name and clicks "Greet".
Example:
```txt
Hello, Niket
```

## Reset Button
Reset button:
- Clears all box colors
- Clears input field
- Resets heading to "Hello"

# Concepts Used
- `querySelector()`
- `querySelectorAll()`
- `addEventListener()`
- `forEach()`
- DOM Manipulation
